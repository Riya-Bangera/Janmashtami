# Admin Staff Management - Quick Reference Guide

## 🚀 Quick Start

### Adding New Staff (Simplified!)
1. Click **"Add Staff"** button (orange button, top right)
2. Enter **Username** and **Password** only
3. Click **"Add Staff Member"**
4. ✅ Done! Staff is created as Judge by default

### Changing Staff Role
1. Click **edit icon** (blue pencil) on staff card
2. Select new role from dropdown:
   - **Admin** - Full system access
   - **Judge** - Can score assigned competitions
   - **Host** - Can manage all participants
3. Click **"Update Staff Member"**

### Assigning Competitions to Judges
**Two-Step Process:**

**Step 1:** Select Age Category
- Click first dropdown (-- CATEGORY --)
- Choose: Kids, Juniors, or Teens

**Step 2:** Select Competition
- Click second dropdown (-- EVENT --)
- Choose from filtered competitions
- ✅ Assignment confirmed with notification

**Remove Assignment:**
- Click **X** on assigned event badge

### Admin Self-Management
✅ You can edit your own credentials!
1. Find your card in the staff list
2. Click edit icon
3. Update username or password
4. Save changes

## 📋 Role Permissions

| Role | Permissions |
|------|-------------|
| **Admin** | Full access to all features, can manage staff, competitions, registrations, and settings |
| **Judge** | Access only to assigned competitions for scoring |
| **Host** | Access to all participants for stage management |

## 💡 Tips

- **Default Role**: All new staff start as Judges
- **Role Changes**: Can be changed anytime through edit
- **Judge Assignments**: Use age category filter to avoid mistakes
- **Host Access**: Hosts don't need competition assignments
- **Multiple Admins**: You can create multiple admin accounts
- **Password Visible**: Passwords are visible in cards for easy reference

## 🎯 Common Tasks

### Creating a Judge
1. Add staff (username + password)
2. Assign competitions using age category filter
3. Done!

### Creating a Host
1. Add staff (username + password)
2. Edit the staff member
3. Change role to "Host"
4. Done!

### Creating Another Admin
1. Add staff (username + password)
2. Edit the staff member
3. Change role to "Admin"
4. Done!

### Updating Your Password
1. Find your own card
2. Click edit
3. Enter new password
4. Save

## ⚠️ Important Notes

- **Delete Carefully**: Deleting staff is permanent
- **Judge Access**: Judges can only see their assigned competitions
- **Host Access**: Hosts can see all participants
- **Admin Access**: Admins can see and manage everything
