## First-Time Setup

After logging in as admin for the first time:

1. **Configure System Settings**
   - Update UPI ID for payment collection
   - Set event year and date
   - Toggle registration status

2. **Set Up Competitions**
   - Add or modify competition events
   - Set competition dates and times
   - Configure scoring rubrics

3. **Add Staff Members**
   - Create judge accounts
   - Create host accounts
   - Assign judges to specific competitions

4. **Monitor Registrations**
   - View participant registrations
   - Process on-spot registrations
   - Export data to Excel
