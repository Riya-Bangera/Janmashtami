## 🎯 Common Tasks

### Calling Participants to Stage
1. Select age category
2. Select competition
3. Find participant in queue
4. Click **"Call to Stage"** button
5. ✅ Participant marked as called

### Viewing Winners
1. Select age category
2. Select competition
3. Click **"Show Winners"** button (if results published)
4. View Divine Ranks celebration screen
5. Click **"Back to Queue"** to return

### Switching Competitions
1. Change age category (competition resets automatically)
2. Select new competition from filtered list
3. View new participant queue

---

## 💡 Tips

- **Age First**: Always select age category before competition
- **Auto-Reset**: Changing age category clears competition selection
- **Filtered List**: Only competitions for selected age group appear
- **Called Status**: Participants show "Called" badge after being called
- **Winners**: Show Winners button only appears when results are published

---

## 🚀 Quick Workflow

```
Login → Select Age Category → Select Competition → Call Participants → Show Winners
```

---

## ⚠️ Important Notes

1. **Competition dropdown is disabled** until you select an age category
2. **Changing age category** automatically resets your competition selection
3. **Winners can only be shown** after judges publish results
4. **All participants** for the selected competition appear in the queue
5. **Called status** persists across page refreshes

## 🔧 Troubleshooting

### Competition dropdown is disabled
- **Solution**: Select an age category first

### No competitions showing
- **Solution**: Make sure competitions are created for the selected age group

### Winners button not showing
- **Solution**: Judges need to publish results first

### Participant not in queue
- **Solution**: Check if participant registered for this competition
